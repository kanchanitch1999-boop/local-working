    เชื่อมต่อ Firebase เพื่อ sync ข้อมูลขึ้น cloud — เปิดจากเครื่องไหนก็ได้ข้อมูลเดิม

    วิธีสร้าง Firebase (ฟรี)
    1. ไปที่ console.firebase.google.com
    2. กด Create a project → ตั้งชื่อ → สร้าง
    3. เมนูซ้าย กด Build → Realtime Database → Create Database → Start in test mode
    4. กด Project settings (ไอคอนเฟือง) → เลื่อนลงหา Your apps → กด </> (Web)
    5. ตั้งชื่อ app → Register → copy ส่วน firebaseConfig = {...}
    6. วางข้อมูล JSON ด้านบน → กด Connect Firebase
